
To run application and unit Tests
    Run console app with the following command 
        dotnet run --project TelstraRobot

    Run Unit tests with the following command
        dotnet test

To see the code files
    Robot library class code file
        TelstraRobot/Robot.cs

    Main running console program code file
        TelstraRobot/Program.cs

    Unit Test code file
        TelstraRobot.Tests/UnitTest1.cs
